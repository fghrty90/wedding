# 경성 ♥ 서희 모바일 청첩장

깔끔하고 모던한 스타일의 단일 HTML 모바일 청첩장입니다.

## 사용법

1. `index.html` 을 더블클릭하면 브라우저에서 바로 열립니다.
2. 사진을 넣으려면 `photos/` 폴더에 아래 이름으로 저장하세요.
   - `photos/main.jpg` — 메인(첫 화면) 배경 사진
   - `photos/1.jpg` ~ `photos/16.jpg` — 갤러리 사진 16장
   - 사진을 넣지 않으면 자동으로 빈 칸(번호)으로 표시됩니다.

## 직접 수정할 부분

| 위치 | 내용 |
|------|------|
| 계좌번호 | `data-copy="..."` 와 화면 표시 `000-0000-0000 · 은행명` 부분 |
| 카카오 공유 | Kakao JavaScript SDK 키 연결 필요 (지금은 안내 토스트만) |
| 방명록 | 현재는 브라우저 로컬 저장(localStorage). 실제 공유 저장이 필요하면 백엔드 연동 필요 |

## 기능

- D-DAY 실시간 카운트다운
- 갤러리 라이트박스(확대/좌우 넘김)
- 신랑·신부 전화 연결
- 계좌번호 원터치 복사
- 네이버/카카오/구글 지도 길찾기
- 방명록 작성
- 링크 복사 공유
- 스크롤 등장 애니메이션

## 배포 (선택)

정적 파일이므로 어디든 올릴 수 있습니다: Netlify, Vercel, GitHub Pages, Cloudflare Pages 등.
폴더 전체(`index.html` + `photos/`)를 그대로 업로드하면 됩니다.
